🔴 الخطوة 1: إنشاء ملف .env
-----------------------------
طريقتان:

أ) تلقائياً (Windows):
   - نقرتين على: CREATE-ENV-FILE.bat

ب) يدوياً:
   1. أنشئ ملف جديد باسم: .env
   2. انسخ المحتوى من: env-template.txt
   3. احفظ الملف


🔴 الخطوة 2: رفع .htaccess على السيرفر
---------------------------------------
1. افتح ملف: api/.htaccess
2. ارفعه إلى: dev.murjan.sa/public_html/.htaccess
   (أو أضف محتواه إلى الملف الموجود)


🔴 الخطوة 3: تشغيل التطبيق محلياً
-----------------------------------
في Terminal:

cd woo-product-manager-main
npm install              ← تم بالفعل ✅
npm run dev             ← شغل التطبيق

افتح المتصفح: http://localhost:8080


🔴 الخطوة 4: النشر على Vercel
-------------------------------
1. اذهب إلى: https://vercel.com/dashboard
2. اختر مشروع: woo-silk
3. Settings → Environment Variables
4. أضف المتغيرات من ملف env-template.txt
5. Redeploy
