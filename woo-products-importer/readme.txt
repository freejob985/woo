A simple plugin to import 40 products (20 Physical and 20 Variable) into WooCommerce.

== Description ==


WooCommerce Products Importer is a powerful plugin that allows you to quickly populate your WooCommerce store with sample products for testing or demonstration purposes.

= Features =

* Import 40 high-quality products with one click
* 20 Physical Products - Tangible items with shipping details
* 20 Variable Products - Products with color and size variations
* Automatic categorization
* Random pricing and stock quantities
* Easy deletion of all imported products
* Clean and intuitive admin interface

= Product Types =

**Physical Products:**
Tangible items that get delivered to customers. Each physical product includes:
* Product name and description
* Pricing information
* Stock management
* Shipping dimensions and weight
* Unique SKU

**Variable Products:**
Products with variations like color or size. Each variable product includes:
* Multiple variations (different colors and sizes)
* Individual pricing per variation
* Stock management per variation
* Unique SKU for each variation

== Installation ==

1. Upload the `woo-products-importer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure WooCommerce is installed and activated
4. Navigate to 'Products Importer' in the WordPress admin menu
5. Click 'Import 40 Products' to add the products to your store

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, WooCommerce must be installed and activated for this plugin to work.

= Can I customize the products? =

Yes, after import, you can edit the products like any other WooCommerce product through the Products menu.

= How do I delete the imported products? =

Simply go to the Products Importer page and click the 'Delete All Imported Products' button.

= Will this affect my existing products? =

No, this plugin only manages the products it creates. Your existing products will not be affected.
